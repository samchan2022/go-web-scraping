Web crawler
===================================================
This project is a web crawler which recursively crawl the all links of the target domain concurrently.

powershell
===================================================
To run 
---------------------------------------------------
`./run.ps1`

To test
---------------------------------------------------
`test.ps1`

Linux
===================================================
To run 
---------------------------------------------------
`./run.sh`

To test
---------------------------------------------------
`test.sh`

Go
===================================================
To run (go lang)
---------------------------------------------------
`go run .`

To test
---------------------------------------------------
`go test -v`
